### Upload File

$\_FILES ជាអឺរេពីរវិមាត្រដែលផ្ទុកព័ត៏មានអំពីត្រូវ upload ដូចជា៖
1. ឈ្មោះ (name)
2. ទីតាំឯកសារត្រូវអាប់ឡោដ full_path
3. ប្រភេទឯកសារ type
4. ទីតាំងបណ្តោះអាសន្នឯកសារបានចម្លងសម្រាប់អាប់ឡោដ tmp_name
5. កំហុស error
6. ទំហំ size។

ឧទាហរណ៍អំពីតម្លៃផ្ទុកក្នុងឈ្មោះសម្គាល់ធាតុអឺរេ $\_FILES

| ឈ្មោះសម្គាល់ធាតុ |            តម្លៃ             |
| ---------------- | :--------------------------: |
| name             |            a.gif             |
| full_path        |          d:\\a.gif           |
| type             |          image/gif           |
| tmp_name         | C:\\wamp64\\tmp\\phpDEE7.tmp |
| error            |              0               |
| size             |            11111             |
អាប់ឡោដឯកសារគឺផ្ទេរឯកសារបានចម្លងសម្រាប់អាប់ឡោដទៅកាន់ទីតាំងបានកំណត់លើ web server
```php
move_uploaded_file($_FILES["file"]["tmp_name"], $អថេរទីតាំងលើម៉ាស៊ីន)
```  
> enctype="multipart/form-data" របស់សំណុំបែបបទបញ្ជាក់ថា ទិន្ន័យក្នុងសំណុំបែបបទត្រូវ encode ចេញជាច្រើនចំណែកដែលចំណែកនីមួយៗមានអត្តសញ្ញាណសម្គាល់។

### Read File From Folder

```php
glob("ទីតាំងនិងលំនាំលក្ខណ្ឌ")
```
